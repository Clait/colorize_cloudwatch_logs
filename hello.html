Colorize CloudWatch Logs

Group visually the logs on AWS CloudWatch

This extension adds color to boundaries of AWS Lambda logs, lines having ANSI terminal codes and makes bold "[REPORT]" and "[ERROR]" logs.

This extension doesn't collect any user and webpage information. It only runs on AWS CloudWatch Logs webpage. It is free to use.

Github project: https://github.com/ilhan-mstf/colorize_cloudwatch_logs
